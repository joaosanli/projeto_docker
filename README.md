# Projeto — Docker & Microsserviços (Python + Flask)

Este repositório contém 5 desafios, cada um com **Dockerfiles**, **scripts** e/ou **docker-compose**,
além de **READMEs** com arquitetura, decisões e instruções de execução.

Estrutura:
```
/desafio1      # Containers em rede
/desafio2      # Volumes e persistência (SQLite)
/desafio3      # Compose com web + db + cache
/desafio4      # Microsserviços independentes
/desafio5      # Microsserviços com API Gateway
```

> Observação: Os exemplos foram escritos para fins didáticos, com foco em clareza e isolamento.
> Sinta-se livre para adaptar nomes de imagens, portas e variáveis.
